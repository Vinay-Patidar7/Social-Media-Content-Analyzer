# Social Media Content Analyzer

A small web app that lets users upload PDFs or images of their social media copy, extracts the text, and gives simple engagement suggestions.

## Tech stack

- Backend: Node.js, Express, Multer, pdf-parse, Tesseract.js
- Frontend: React (Vite)


### Backend

```bash
cd server
cp .env.example .env # or create .env
npm install
npm run dev
